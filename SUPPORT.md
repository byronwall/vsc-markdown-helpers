# Support

If you hit a bug, unexpected rendering issue, or packaging problem:

- Open an issue at https://github.com/byronwall/vsc-md-helpers/issues
- Include the VS Code version, extension version, and your operating system.
- Attach a minimal markdown sample or repository layout that reproduces the problem.

For local debugging, run:

```bash
pnpm install
pnpm run compile
pnpm run dev:host
```
